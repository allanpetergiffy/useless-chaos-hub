# Useless Chaos Hub Website

A fun, interactive, and completely pointless website built with HTML, CSS, and JavaScript.  
Features include:
- Infinite spinning buttons
- Random excuse generator
- Fake progress bar
- Runaway text
- Random background colors
- Funny sound effects
- Fake AI chatbot

## How to Run
1. Clone the repository
2. Open `index.html` in your browser
3. or click this link
4. https://allanpetergiffy.github.io/useless-chaos-hub/
